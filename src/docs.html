SmartCalc v1.0 by Umaradri

Научный калькулятор с возможностями:

  • Вычисление произвольных скобочных арифметических выражений в инфиксной нотации
  • Вычисление произвольных скобочных арифметических выражений в инфиксной нотации с подстановкой значения переменной x в виде числа
  • Построение графика функции, заданной с помощью выражения в инфиксной нотации с переменной x

    • Поддерживаемые операторы и функции

    • Скобки
    • Сложение
    • Вычитание
    • Умножение
    • Деление
    • Возведение в степень
    • Остаток от деления
    • Унарный плюс
    • Унарный минус
    • Косинус
    • Синус
    • Тангенс
    • Арккосинус
    • Арксинус
    • Арктангенс
    • Квадратный корень
    • Натуральный логарифм
    • Десятичный логарифм

    А также со специальным режимом "кредитный калькулятор" с возможностями расчета:

    • Ежемесячного платежа
    • Переплаты по кредиту
    • Общей выплаты

    Сборка и запуск

    Для установки и запуска калькулятора введите команду make install

    Удаление

    Для удаления калькулятора используйте команду make uninstall
